### Apple Box - A QT based gui pip installer ###
Hello, and welcome to Apple box! the qt based gui pip installer! This python pip tool is designed to make it easier for python devs to manage their pip packages. Some of the ways that make it handy for this are:
- Installing mulitple versions of python packages from pipy
- Vewing pip install logs
- Creating virtual enviorments for your projects
- Making removing and updating pip packages much easier via a GUI
